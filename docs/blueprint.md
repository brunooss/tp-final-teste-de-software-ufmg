# **App Name**: Clarity Compass

## Core Features:

- Yes/No Decision Tool: Allows the user to input a decision context and choose between 'Yes' or 'No' options. A generative AI tool gives advice in making the decision.
- Multiple Choice Decision Tool: Users define the decision context and input multiple options to select from. A generative AI tool gives advice on how to make the decision.
- Financial Decision Tool (Fixed + Variable Costs): Analyzes financial decisions based on fixed and variable costs, aiding users in financial planning. A generative AI tool suggests various weights for each of the inputs.
- Financial Spending Decision Tool: Compares financial spending options (e.g., financing vs. cons√≥rcio), helping users make informed decisions.
- Decision History: Maintained history of all past decisions made within the app.

## Style Guidelines:

- Primary color: Soft green (#A7D1AB) to evoke calmness and clarity.
- Background color: Off-white (#F5F5F5) to provide a clean and neutral backdrop.
- Accent color: Light beige (#E5D7C0) for secondary elements and subtle highlights.
- Body and headline font: 'Inter', a sans-serif, providing a modern and objective look.
- Use simple, clean icons in a consistent style to represent different decision types and categories.
- Clean and intuitive layout with clear sections for each decision-making tool.
- Subtle transitions and animations to guide the user through the decision-making process.
